========================================
    云农谷引流原型设计 - 使用说明
========================================

📱 项目简介
-----------
这是一套完整的小程序流量运营原型设计，用于展示如何将机井平台的用户引流至云农谷平台。
所有页面均为HTML静态页面，不涉及真实后端逻辑，仅用于产品演示和评审。


🚀 快速开始
-----------
1. 在浏览器中打开 index.html 文件
2. 点击对应的功能模块卡片进入各个页面
3. 体验不同的引流场景和交互流程


📂 项目结构
-----------
云农谷引流原型/
├── index.html                          # 主导航页（从这里开始）
├── README.md                           # 详细的项目文档
├── 使用说明.txt                        # 本文件
└── pages/                              # 所有功能页面
    ├── jijing-home.html               # 机井首页（带Banner引流）
    ├── repair-success.html            # 报修成功页（带AI入口）
    ├── jijing-detail.html             # 机井详情页（带气象入口）
    ├── yunnongu-landing.html          # 云农谷落地页
    ├── material-success.html          # 资料领取成功页
    ├── ai-diagnosis.html              # AI病虫害识别
    ├── weather-detail.html            # 天气详情
    ├── share-demo.html                # 分享功能演示
    ├── share-landing.html             # 分享落地页
    └── feature-matrix.html            # 功能分享矩阵


🎯 三大功能模块
---------------

【模块一：首页资源位引流】
  - 通过免费资料包吸引用户注册云农谷
  - 页面：机井首页 → 云农谷落地页 → 资料领取成功

【模块二：场景化后链路引流】
  - 在报修、查看机井等场景推荐相关服务
  - 页面：报修成功页/机井详情页 → AI识别/天气详情

【模块三：分享与裂变矩阵】
  - 核心功能支持分享，实现社交传播
  - 页面：分享功能演示 → 分享落地页 → 功能页


💡 体验建议
-----------
1. 按照模块顺序依次体验各个页面
2. 注意页面间的跳转逻辑和参数传递
3. 体验分享功能和弹窗交互
4. 查看控制台了解URL参数（用于数据追踪）


📊 核心指标
-----------
- 引流点击率目标：≥ 15%
- 跳转转化率目标：≥ 30%
- 功能渗透率目标：≥ 50%


⚠️  注意事项
------------
1. 这是静态原型，所有数据都是模拟的
2. 交互会触发弹窗提示，说明业务逻辑
3. 不包含真实的后端接口和微信API
4. 建议使用Chrome/Safari/Edge浏览器


📞 技术说明
-----------
- 使用纯HTML+CSS+JavaScript实现
- 采用手机壳样式模拟真实小程序
- 支持响应式交互（hover、点击效果）
- 所有页面可独立访问
