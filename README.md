# 云农谷引流原型设计

## 📋 项目概述

这是一套完整的小程序流量运营原型设计，展示了如何将**机井管护平台**的2万+存量用户低成本引流至**云农谷**平台，并通过社交裂变实现二次增长。

**文档版本：** V1.0  
**创建日期：** 2025-11-18  
**设计目标：** 低成本引流 + 高转化率 + 社交裂变

---

## 🎯 核心功能模块

### 模块一：首页资源位引流
通过高价值资料包吸引用户主动跳转到云农谷

**页面列表：**
- `pages/jijing-home.html` - 机井首页（带Banner引流入口）
- `pages/yunnongu-landing.html` - 云农谷落地页（资料包领取）
- `pages/material-success.html` - 资料领取成功页

**引流策略：**
- ✅ 利益诱导：免费《机井管护手册》+ 实操视频
- ✅ 无感跳转：一键授权，无需重复注册
- ✅ 即时满足：注册后立即可查看/下载资料

---

### 模块二：场景化后链路引流
在用户使用机井服务的关键节点提供延伸服务

**页面列表：**
- `pages/repair-success.html` - 报修成功页（带AI服务入口）
- `pages/jijing-detail.html` - 机井详情页（带气象信息入口）
- `pages/ai-diagnosis.html` - AI病虫害识别页
- `pages/weather-detail.html` - 天气详情页

**引流策略：**
- ✅ 场景延伸：报修后推荐AI诊断作物
- ✅ 需求关联：查看机井时提供灌溉建议
- ✅ 参数透传：自动带入地理位置信息

---

**页面列表：**

## 🚀 快速开始

### 使用方法

1. **打开主导航页**
   ```
   直接在浏览器中打开 index.html
   ```

2. **浏览各个模块**
   - 点击对应模块卡片查看不同的功能页面
   - 所有页面都采用手机壳样式，模拟真实小程序体验

3. **体验交互流程**
   - 点击Banner/卡片会触发跳转（带参数透传）
   - 部分功能会显示弹窗提示说明业务逻辑
   - 所有交互都不涉及真实后端接口

## 🎨 设计特点

### 1. 视觉设计
- ✅ 手机壳样式模拟真实小程序环境
- ✅ 统一的色彩体系（绿色主调，符合农业主题）
- ✅ 渐变色卡片增强视觉吸引力
- ✅ 响应式交互效果（hover、active状态）

### 2. 用户体验
- ✅ 流程闭环：每个引流点都有明确的落地页
- ✅ 即时反馈：所有交互都有弹窗或页面跳转提示
- ✅ 参数透传：模拟真实的数据传递（地理位置、来源标识等）
- ✅ 降低门槛：一键授权，无需复杂注册流程

### 3. 业务逻辑
- ✅ 利益驱动：资料包、免费服务吸引用户
- ✅ 场景关联：在用户需要时推荐相关功能
- ✅ 社交裂变：核心功能支持分享，配置独立文案
- ✅ 数据追踪：所有入口都携带来源参数（便于后续分析）

## 📊 数据埋点说明

虽然这是静态原型，但在实际实现时需要在以下关键节点添加埋点：

### 来源端（机井平台）
- `well_home_banner_click` - 首页Banner点击
- `well_repair_success_ai_click` - 报修成功页AI入口点击
- `well_detail_weather_click` - 机井详情页气象入口点击

### 目标端（云农谷）
- `app_launch_source` - 启动来源统计
- `conversion_register` - 新增注册转化
- `feature_activation` - 功能使用激活

### 分享裂变
- `share_click` - 分享按钮点击
- `share_success` - 分享成功次数
- `share_landing_visit` - 分享落地页访问
- `share_conversion` - 分享带来的注册转化

## 💡 实现建议

### 技术对接
1. **UnionID 互通**
   - 确保机井平台和云农谷在同一微信开放平台账号下
   - 实现用户无感登录

2. **参数透传**
   ```javascript
   // 示例：机井首页跳转云农谷
   wx.navigateToMiniProgram({
     appId: 'yunnongu_appid',
     path: 'pages/landing/index',
     extraData: {
       source: 'jijing_home_banner',
       location: '鄢城镇丁标村',
       user_id: 'xxx'
     }
   })
   ```

3. **小程序码生成**
   - 为每个分享功能页生成独立小程序码
   - 携带 scene 参数用于追踪来源

### 内容运营
1. **资料包制作**
   - 将《操作手册》拆分为多个易读的PDF
   - 录制3-5分钟的实操视频（手机可播放）
   - 准备高频问题的图文解答

2. **分享文案优化**
   - A/B测试不同文案的转化率
   - 结合时令农事更新文案（如"小麦返青期""玉米拔节期"）
   - 增加紧迫感（"限时免费""仅剩100份"）

3. **公众号内容**
   - 每周发布1-2篇农技文章
   - 文中/文末插入小程序卡片
   - 引导用户扫码体验功能

## ⚠️ 注意事项

1. **这是静态原型**
   - 所有数据都是模拟数据
   - 不包含真实的后端接口
   - 仅用于产品演示和评审

2. **交互说明**
   - 点击交互元素会触发 `alert` 提示
   - 跳转会携带URL参数（可在控制台查看）
   - 部分功能用弹窗模拟真实流程

3. **浏览器兼容**
   - 建议使用现代浏览器（Chrome、Safari、Edge）
   - 移动设备上也可以正常访问
   - 最佳观看宽度：375px（iPhone X尺寸）
